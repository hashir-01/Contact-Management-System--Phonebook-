# 📱 Phonebook / Contact Management System

A **C++-based Contact Management System** developed to demonstrate the practical implementation of **Data Structures** in solving real-world problems involving contact organization, searching, privacy, and dynamic data handling.

## 🚀 Features

* Add, search, update, and delete contacts
* Public & private contact categorization
* PIN-protected access for private contacts
* Contact blocking & unblocking system
* Call simulation and call history management
* Fast and organized contact retrieval

## 🛠 Data Structures Used

### 1. Binary Search Tree (BST)

Used for:

* Efficient searching of contacts
* Sorted storage of contact data
* Fast insertion and traversal

### 2. Doubly Linked List

Used for:

* Managing blocked contacts
* Efficient insertion and deletion
* Forward and backward traversal

## 💻 Technologies Used

* **Language:** C++
* **Concepts:** Data Structures, Recursion, Modular Programming
* **Platform:** Console-Based Application

## 🎯 Project Objective

To design a **secure, efficient, and user-friendly contact management system** while applying fundamental data structure concepts to a practical implementation.

## 📌 Learning Outcomes

This project strengthened understanding of:

* Binary Search Trees (BST)
* Doubly Linked Lists
* Dynamic data handling
* Recursion & traversal techniques
* Secure system design
* Real-world implementation of Data Structures
